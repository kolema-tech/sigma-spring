# 注釋 & 工具

## cc

變量：
```
CREAT_DATE date()
CREAT_TIME time()
```

Template：
```java
/**
 * @author zen peng.
 * @version 1.0
 * date-time: $CREAT_DATE$-$CREAT_TIME$
 * desc:
 **/
```

## mc

變量：
```
METHOD_PARAM methodParameters()
```

Template:
```java
/**
 * param: $METHOD_PARAM$
 **/
```
